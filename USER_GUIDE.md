# Руководство пользователя

Инструкция по работе с интеллектуальной поисковой системой по внутренней базе знаний университета.

## 1. Регистрация

1. Откройте приложение по адресу `http://localhost:5173`
2. Если вы новый пользователь — вы автоматически попадёте на страницу **регистрации**
3. Введите логин (от 3 символов) и пароль (не менее 6 символов), подтвердите пароль
4. Нажмите **«Зарегистрироваться»**

После успешной регистрации система автоматически выполнит вход и перенаправит вас на главную страницу.

## 2. Вход в систему

Если у вас уже есть аккаунт:
1. Перейдите по ссылке **«Войти»** на странице регистрации (или откройте `http://localhost:5173/login`)
2. Введите логин и пароль
3. Нажмите **«Войти»**

## 3. Загрузка документов

На главной странице слева расположена панель **«Загрузка»**:

1. **Перетащите** файл (или несколько файлов сразу) в область с пунктирной рамкой — либо **нажмите** на неё, чтобы открыть системный диалог выбора файлов
2. Поддерживаются форматы **PDF** и **DOCX**, размер файла — до **20 МБ**
3. Во время загрузки под областью загрузки появится строка с именем файла и статусом:
   - **«Загрузка...»** — файл передаётся на сервер
   - **«Индексация...»** — сервер извлекает текст и индексирует документ
   - **«Готово»** — документ успешно загружен и доступен для поиска
   - **«Ошибка»** — что-то пошло не так (неверный формат, слишком большой размер, повреждённый файл); текст ошибки отображается под именем файла

4. После успешной загрузки документ появится в списке **«Документы»** ниже, с указанием даты загрузки и количества найденных текстовых фрагментов

## 4. Поиск по документам

Справа расположена панель **«Поиск»**:

1. Введите поисковый запрос в поле ввода
2. Запустите поиск **кнопкой «Найти»** или нажатием клавиши **Enter**
3. Результаты появятся в виде карточек ниже. Каждая карточка содержит:
   - Название файла и номер страницы, где найдено совпадение
   - Фрагмент текста с **подсветкой** совпавших слов жёлтым цветом
   - Индикатор релевантности (шкала из полосок + числовое значение) — чем выше, тем точнее совпадение

4. Если результатов больше 10 — под списком появится **пагинация** для перехода между страницами
5. Если по запросу ничего не найдено, появится сообщение: *«По вашему запросу ничего не найдено. Попробуйте изменить формулировку»*

## 5. Выход из системы

Кнопка **«Выйти»** расположена в правом верхнем углу главной страницы, рядом с вашим именем пользователя.

## Частые вопросы

**Почему документ показывает статус «Ошибка»?**
Проверьте, что файл имеет формат PDF или DOCX, весит не более 20 МБ и не повреждён.

**Почему поиск не находит документ, который я точно загрузил?**
Убедитесь, что документ полностью проиндексировался (статус «Готово» в списке документов). Также поиск ищет по содержимому текста — если документ состоит только из изображений (сканы без текстового слоя), текст не будет извлечён.

**Можно ли загружать несколько файлов одновременно?**
Да — выделите сразу несколько файлов в диалоге выбора или перетащите группу файлов в область загрузки.
